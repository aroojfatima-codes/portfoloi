## Installation

Clone the repository and install dependencies:

```bash
git clone <repository-url>
cd portfolio
npm install
npm run dev
```

The development server will start locally. Open the URL shown in the terminal, usually:

```bash
http://localhost:3000
```

Build the project for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

---

## Deployment

This portfolio can be deployed easily on **Vercel**.

1. Push the project to GitHub.
2. Open [Vercel](https://vercel.com/).
3. Import the GitHub repository.
4. Set the build command:

```bash
npm run build
```

5. Set the output directory:

```bash
dist
```

6. Add required environment variables if using the contact form email integration.
7. Deploy the project.
